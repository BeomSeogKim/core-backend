# 비트 연산 (Bit Manipulation)

## 핵심 개념

> [!note] 핵심 요약
> 비트 연산은 정수를 이진수 레벨에서 직접 조작하는 기법이다. **XOR**, **AND**, **OR**, **Shift** 등을 활용하면 반복문 없이 O(1)로 특정 연산을 수행할 수 있다.

### 주요 연산자

| 연산자 | 기호 | 설명 | 예시 |
|--------|------|------|------|
| AND | `&` | 둘 다 1이면 1 | `5 & 3` = `1` |
| OR | `\|` | 하나라도 1이면 1 | `5 \| 3` = `7` |
| XOR | `^` | 서로 다르면 1 | `5 ^ 3` = `6` |
| NOT | `~` | 비트 반전 | `~5` = `-6` |
| Left Shift | `<<` | 왼쪽으로 이동 (×2) | `5 << 1` = `10` |
| Right Shift | `>>` | 오른쪽으로 이동 (÷2) | `5 >> 1` = `2` |

## 동작 원리

### 이진수 변환

십진수를 2로 나누면서 나머지를 모은다:

```
41 → 41 % 2 = 1, 20 % 2 = 0, 10 % 2 = 0, 5 % 2 = 1, 2 % 2 = 0, 1 % 2 = 1
   → 101001 (역순으로 읽기)
```

```java
// 십진수 → 이진수 각 자릿수 순회
while (N > 0) {
    int bit = N % 2;  // 현재 자릿수
    N /= 2;           // 다음 자릿수로
}
```

### XOR 핵심 성질

```
a ^ a = 0    // 같은 값 두 번 XOR하면 0
a ^ 0 = a    // 0과 XOR하면 자기 자신
a ^ b ^ a = b  // 순서 상관없이 짝은 상쇄
```

## 코드 예시

### 짝 없는 원소 찾기 (XOR)

배열에서 하나만 홀수 번 등장하는 원소를 O(N) 시간, O(1) 공간으로 찾는다:

```java
public int findUnpaired(int[] A) {
    int result = 0;
    for (int a : A) {
        result ^= a;  // 짝수 번 등장하는 값은 상쇄
    }
    return result;
}
```

```
[3, 5, 3] → 0 ^ 3 = 3 → 3 ^ 5 = 6 → 6 ^ 3 = 5 ✅
```

### Binary Gap (이진수에서 1 사이 최대 간격)

```java
public int binaryGap(int N) {
    int maxGap = 0;
    int gap = 0;
    boolean found = false;

    while (N > 0) {
        int bit = N % 2;
        N /= 2;
        if (bit == 1) {
            if (found) {
                maxGap = Math.max(maxGap, gap);
            }
            found = true;
            gap = 0;
        } else {
            gap++;
        }
    }
    return maxGap;
}
```
