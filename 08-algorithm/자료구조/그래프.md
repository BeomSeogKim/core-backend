# 그래프 + DFS / BFS

## 핵심 개념

**그래프(Graph)**는 노드(정점, Vertex)와 간선(Edge)으로 이루어진 자료구조다. Tree와 달리 **순환(Cycle)이 가능**하고, 방향/가중치의 유무에 따라 다양한 형태를 가진다. **DFS**와 **BFS**는 그래프를 탐색하는 두 가지 대표적인 방법이다.

| 종류 | 설명 |
|------|------|
| **무방향 그래프** | 간선에 방향 없음 (양방향) |
| **방향 그래프** | 간선에 방향 있음 (단방향) |
| **가중치 그래프** | 간선에 비용/거리 값이 있음 |
| **순환 그래프** | 사이클이 존재 |
| **비순환 그래프 (DAG)** | 방향 + 사이클 없음 (위상 정렬 가능) |

### Graph vs Tree

| | Graph | Tree |
|---|---|---|
| 순환 | **가능** | 불가 |
| 루트 | 없음 | 있음 |
| 부모-자식 | 없음 | 있음 |
| 간선 수 | 자유 | N-1개 (노드 N개) |
| 관계 | Tree는 Graph의 부분집합 | 순환 없는 연결 그래프 |

### 그래프 표현 방식

#### 인접 행렬 (Adjacency Matrix)

```
    A  B  C  D
A [ 0, 1, 1, 0 ]
B [ 1, 0, 0, 1 ]
C [ 1, 0, 0, 1 ]
D [ 0, 1, 1, 0 ]

공간: O(V²)
간선 확인: O(1)
모든 인접 노드 탐색: O(V)
```

#### 인접 리스트 (Adjacency List)

```
A → [B, C]
B → [A, D]
C → [A, D]
D → [B, C]

공간: O(V + E)
간선 확인: O(degree)
모든 인접 노드 탐색: O(degree)
```

| | 인접 행렬 | 인접 리스트 |
|---|---|---|
| 공간 | O(V²) | **O(V + E)** |
| 간선 확인 | **O(1)** | O(degree) |
| 적합 | 밀집 그래프 (간선 많음) | **희소 그래프 (간선 적음)** |

실무에서는 대부분 **인접 리스트**를 사용한다.

### DFS (Depth-First Search, 깊이 우선 탐색)

한 방향으로 **끝까지** 들어간 뒤, 더 이상 갈 곳 없으면 돌아와서 다른 경로 탐색. **Stack**(또는 재귀)을 사용한다.

<div style="background: #1e1e2e; border-radius: 8px; padding: 12px;">
<div style="color: #89b4fa; font-weight: bold; margin-bottom: 8px;">DFS 탐색 순서</div>
<div style="color: #cdd6f4; line-height: 1.8;">
<pre style="margin: 0; color: #cdd6f4;">
        <span style="color: #f38ba8;">A</span> ①
       / \
      <span style="color: #f9e2af;">B</span> ②  <span style="color: #a6e3a1;">C</span> ⑤
     / \     \
    <span style="color: #89b4fa;">D</span> ③ <span style="color: #fab387;">E</span> ④  <span style="color: #cba6f7;">F</span> ⑥
</pre>
<div style="margin-top: 8px; background: #45475a; padding: 6px 10px; border-radius: 4px;">
방문 순서: <span style="color: #f38ba8;">A</span> → <span style="color: #f9e2af;">B</span> → <span style="color: #89b4fa;">D</span> → <span style="color: #fab387;">E</span> → <span style="color: #a6e3a1;">C</span> → <span style="color: #cba6f7;">F</span>
</div>
<div style="margin-top: 4px; color: #6c7086; font-size: 11px;">깊이를 먼저 탐색 → B의 자식(D, E)을 먼저 방문한 후 C로</div>
</div>
</div>
</div>

```java
// DFS — 재귀 구현
void dfs(int node, boolean[] visited, List<List<Integer>> graph) {
    visited[node] = true;
    System.out.print(node + " ");

    for (int next : graph.get(node)) {
        if (!visited[next]) {
            dfs(next, visited, graph);  // 재귀로 깊이 탐색
        }
    }
}
```

### BFS (Breadth-First Search, 너비 우선 탐색)

현재 노드에서 **가까운 것부터** 레벨 단위로 탐색. **Queue**를 사용한다.

<div style="font-family: -apple-system, monospace; max-width: 480px; margin: 12px auto; padding: 16px; font-size: 12px;">
<div style="background: #1e1e2e; border-radius: 8px; padding: 12px;">
<div style="color: #89b4fa; font-weight: bold; margin-bottom: 8px;">BFS 탐색 순서</div>
<div style="color: #cdd6f4; line-height: 1.8;">
<pre style="margin: 0; color: #cdd6f4;">
        <span style="color: #f38ba8;">A</span> ①         ← Level 0
       / \
      <span style="color: #f9e2af;">B</span> ② <span style="color: #a6e3a1;">C</span> ③     ← Level 1
     / \     \
    <span style="color: #89b4fa;">D</span> ④ <span style="color: #fab387;">E</span> ⑤  <span style="color: #cba6f7;">F</span> ⑥  ← Level 2
</pre>
<div style="margin-top: 8px; background: #45475a; padding: 6px 10px; border-radius: 4px;">
방문 순서: <span style="color: #f38ba8;">A</span> → <span style="color: #f9e2af;">B</span> → <span style="color: #a6e3a1;">C</span> → <span style="color: #89b4fa;">D</span> → <span style="color: #fab387;">E</span> → <span style="color: #cba6f7;">F</span>
</div>
<div style="margin-top: 4px; color: #6c7086; font-size: 11px;">가까운 것 먼저 → 같은 레벨(B, C)을 먼저 방문한 후 다음 레벨로</div>
</div>
</div>
</div>

```java
// BFS — Queue 구현
void bfs(int start, List<List<Integer>> graph) {
    boolean[] visited = new boolean[graph.size()];
    Queue<Integer> queue = new LinkedList<>();

    visited[start] = true;
    queue.offer(start);

    while (!queue.isEmpty()) {
        int node = queue.poll();
        System.out.print(node + " ");

        for (int next : graph.get(node)) {
            if (!visited[next]) {
                visited[next] = true;
                queue.offer(next);  // 인접 노드를 큐에 추가
            }
        }
    }
}
```
