# [Terraform](https://developer.hashicorp.com/terraform)
클라우드 인프라를 코드로 관리할 수 있게 해주는 도구

서버, 네트워크, 데이터베이스 같은 인프라를 `코드`로 작성하고 자동으로 생성 / 수정 / 삭제할 수 있다.

### 주요 특징
Terraform은 선언적 방식으로 동작한다.

**무엇**을 원하는지 정의하면, Terraform이 알아서 그 상태를 만들어 준다.

멀티 클라우드를 지원하기에 AWS, Azure, GCP등 다양한 클라우드 프로바이더를 하나의 도구로 관리할 수 있다.

### 주요 명령어
- terraform init : 프로젝트 초기화
- terraform plan : 변경사항 미리 보기
- terraform apply : 인프라 적용
- terraform destory : 인프라 삭제
