# Class

## 클래스, 객체, 인스턴스
클래스
- 객체를 만들기 위한 설계도, 틀, 템플릿
- 속성(필드)과 행위(메서드)를 정의
- 코드 상에 존재하는 개념적 존재
- 메모리에 딱 한번만 로드 (Method Area / Metaspace)
```java
public class Car {
  String model;
  int speed;

  void accelerate() { }
}
```

객체
- 클래스를 기반으로 실체화된 것 전체를 지칭하는 포괄적 용어
- 소프트웨어 세계에 구현할 대상
- 클래스의 타입으로 선언된 것
- 실체 + 개념 모두 포함하는 가장 넓은 의미

인스턴스
- 클래스를 통해 실제 메모리에 생성된 실체
- `new` 키워드로 힙 메모리에 할당된 구체적 존재
- _특정 클래스의 객체라는 관계를 강조할 때 사용_
- 'Car 클래스의 인스턴스'처럼 소속 관계 표현

```java
Car myCar = new Car();
// myCar는 참조 변수
// new Car()로 생성된 것이 인스턴스
// 이 실체를 객체라고도 부름
```
