# 접근 제어자 (Access Modifier)

private
- 같은 클래스 내에서만 접근 가능
- 가장 제한적인 접근 수준
- 캡슐화의 핵심 (외부에서 내부 구현 숨김)

default(package-private)
- 접근 제어자를 명시하지 않은 경우
- 같은 패키지 내에서만 접근 가능
- 패키지 외부에서는 접근 불가

protected
- 같은 패키지 + 다른 패키지의 자식 클래스에서 접근 가능
- 상속 관계를 고려한 접근 수준

public
- 어디서든 접근 가능
- 가장 개방적인 접근 수준
- API로 공개할 부분에만 사용
