**1bit와 2수(== 디지털)**
- 1bit: '전기 스위치 1개'
  - 1: 전기가 흐르는 `on`상태
  - 0: 전기가 흐르지 않는 `off`상태

컴퓨터에서 주로 사용하는 단위수는 16진수로, 주로 다음과 같이 표기하여 사용한다.

`0xF4` : 244 (10), (4bit + 4bit = 8bit)

**1 byte**
- 8개의 bit를 하나로 묶은 단위이다.
- 영문자 한 글자가 저장될 수 있는 메모리 크기
- **관리의 최소 단위**

**컴퓨터에서 글자를 다루는 법**

ASCII (American Standard Code for Information Interchange)
> 미국에서 사용하는 표준 코드 체계

영문 대문자 `A` (65(10), 0x41)

**컴퓨터가 사진을 다루는 방법**
- 화소(Pixel): 모니터 화면 상 '점' 하나.
- 여러 점들을 모아 사진을 이룰 수 있음.
- 화소 하나를 표현하는데 8bit, 16bit, 24bit (RGB), 32bit (RGBA)가 필요할 수 있다.
